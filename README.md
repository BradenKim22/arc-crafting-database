# arc-crafting-database
A database systems class project modeling a game's item inventory, crafting recipes, and item transformations to reach your goal.

## Software Used
- MySQL Server 8+
- MySQL Workbench
- ERDPlus
- VS Code
- GitHub
  
## How to Run
1. Open MySQL Workbench
2. Run 01_schema.sql
3. Run 02_seeds.sql

## How to Reset Database:
1) Run 03_delete.sql
2) Re-run 02_seeds.sql
